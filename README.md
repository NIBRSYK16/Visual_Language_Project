# 学术文献全景分析系统

一个基于 React + D3.js 的交互式学术文献数据可视化分析平台。

## 功能特性

- **多维度可视化**：全球科研产出地图、动态词云、作者合作网络、关键词演化、会议趋势、引用关系
- **视图联动**：支持年份筛选、关键词筛选、国家筛选等多种联动方式
- **现代UI设计**：基于 Ant Design 5.x，界面美观易用
- **高性能**：优化的数据处理和渲染性能
- **交互搜索**：支持作者搜索、关键词筛选等

## 快速开始

### 安装依赖

```bash
yarn install
```

### 启动开发服务器

```bash
yarn start
```

### 构建生产版本

```bash
yarn build
```

## 数据准备

将论文数据放置在 `public/data/papers.json`，数据格式请参考 `使用说明.md`。
