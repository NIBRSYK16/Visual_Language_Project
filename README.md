# 学术文献分析可视化

交互式学术文献数据可视化分析页面，支持多维度数据展示和交互分析。

## 已集成功能

- **多维度可视化**：全球科研产出地图、动态词云、3D作者合作网络、关键词演化、国家演化、机构演化、会议趋势、引用关系、3D关键词球形等
- **视图联动**：支持年份筛选、关键词筛选、国家筛选等多种联动方式
- **演化分析**：支持关键词、国家、机构三个维度的演化趋势分析，可按选定年份区间播放
- **交互搜索**：支持作者搜索、关键词筛选、论文列表搜索等
- **3D可视化**：3D作者合作网络和3D关键词球形展示
- **深色主题**：现代化的深色主题界面设计
- **鼠标特效**：流畅的鼠标轨迹特效

## 快速开始

### 安装依赖

```bash
yarn install
```

### 启动开发服务器

```bash
yarn start
```

访问 http://localhost:8000 查看应用

### 构建生产版本

```bash
yarn build
```

构建后的文件在 `dist` 目录

将论文数据放置在 `public/data/papers.json`，数据格式请参考 [使用说明.md](./使用说明.md)。

## 技术栈

- **前端框架**: React 17.x + TypeScript 4.x
- **构建工具**: UmiJS 3.x
- **UI组件库**: Ant Design 5.x
- **数据可视化**: D3.js 7.x + d3-cloud
- **3D渲染**: Three.js
- **样式**: Less

## 主要功能模块

1. **全球科研产出地图** - 展示各国科研产出分布
2. **动态交互词云** - 关键词频率可视化，支持点击筛选
3. **顶会趋势分析** - 各会议论文数量趋势
4. **会议占比分析** - 会议论文占比饼图
5. **作者合作网络（3D）** - 3D可视化作者合作关系
6. **关键词演化图谱** - 关键词随时间的演化趋势
7. **国家论文发表数演化** - 各国论文发表数随时间的变化
8. **机构论文发表数演化** - 各机构论文发表数随时间的变化
9. **文献引用瀑布** - 论文引用关系可视化
10. **3D关键词球形** - 3D球形展示关键词关系
11. **论文列表** - 详细的论文列表，支持搜索和排序

## 特色功能

- **全局播放控制** - 一键同步播放三个演化模块
- **智能筛选联动** - 多个视图模块智能联动更新
- **论文列表搜索** - 支持按标题、作者、机构搜索
- **深色主题** - 护眼的深色主题设计
